The application allows users to securely upload documents and query them using AI.

Authentication is implemented using Spring Security, JWT tokens, and OTP verification.

Uploaded documents are processed through a RAG pipeline where documents are chunked, embeddings are generated using OpenAI embedding models, and vectors are stored in a vector database.

When users ask questions, semantic similarity search retrieves relevant chunks, which are passed to the LLM for grounded answer generation.

The frontend was built using React and Material UI with protected routes, dashboard layouts, upload workflows, and a ChatGPT-style interface.
